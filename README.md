# 🐍 Jeu du Serpent - Javascript

Un mini-jeu classique du **Snake**, développé en **HTML, CSS et JavaScript pur**.  
Simple, rapide, fun et 100% fait maison 🧠💻

---

## 🎮 Objectif du jeu

- Dirige le serpent avec les flèches du clavier
- Mange les pommes pour grandir 🍎
- Ne te mords pas la queue 🐍❌
- Et surtout… bats ton record !

---

## 🚀 Technologies utilisées

- 🧱 **HTML5** – structure de la page
- 🎨 **CSS3** – petit code mais peut être nettement amélioré
- 🧠 **JavaScript** – toute la logique du jeu
